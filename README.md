## Getting Started

### 1. Get Your API Key

Sign up at [console.videodb.io](https://console.videodb.io) to get your free API key with $20 in credits - no credit card required!

### 2. Install VideoDB

```bash
pip install videodb
```

### 3. Install VideoDB Skills (for AI Agents)

```bash
npx skills add video-db/skills
```

### 4. Run the Notebooks

Open any notebook in Google Colab or Jupyter and follow along!

---

## What You'll Learn

### See, Understand, Act Framework

- **SEE**: Ingest video from files, URLs, or live streams
- **UNDERSTAND**: Index content with AI (transcription + visual analysis)
- **ACT**: Search, edit, compile, and automate actions

### Key Features Covered

- Video upload and instant streaming
- Spoken word indexing and keyword search
- Visual scene indexing and semantic search
- AI-powered video editing with Timeline Editor
- Live RTSP stream connection
- Real-time event detection and WebSocket alerts
- Multi-camera synchronization and composition
